Python Flow Visualizer

`app.py`는 진입점만 두고, 기능은 분리된 파일에서 처리합니다.

현재 실행 코드

실행 중인 줄을 강조하고, 편집 모드에서는 입력 코드를 직접 작성합니다.

step 0 / 0 func module line -
실행 기록이 생성되면 코드 흐름이 여기에 표시됩니다.
실행 기록이 생성되면 프레임 흐름이 여기에 표시됩니다.

출력

선택한 step 시점까지의 stdout과 오류를 보여줍니다.

IDLE
실행 출력이 여기에 표시됩니다.

주 시각화

코드에서 감지한 자료구조를 기준으로 가장 알맞은 시각화를 자동 선택합니다.

대기 SUMMARY
실행 결과가 여기에 표시됩니다.

코드 설명

현재 step 설명과 자동 판단된 자료구조를 함께 정리합니다.

실행을 시작하면 설명이 여기에 표시됩니다.
